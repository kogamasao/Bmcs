# 基本設計書

## 1. システム概要
野球チーム・スコア管理システム (Bmcs)
チームの管理、メンバーの管理、試合のスコア入力、成績集計を行うWebアプリケーション。
ASP.NET Core Razor Pagesを使用し、Entity Framework Coreによるデータアクセスを行う。

## 3. 機能一覧

| 機能ID | 機能名 | 概要 | 関連画面 |
| --- | --- | --- | --- |
| F001 | ログイン・ログアウト | ユーザー認証を行う。 | G002 |
| F002 | ユーザー登録 | 新規ユーザーアカウントを作成する。 | G002, G090 |
| F010 | チーム管理 | チーム情報の登録、編集、削除を行う。 | G010-G014 |
| F020 | メンバー管理 | 選手情報の登録、編集、削除を行う。 | G020-G024 |
| F030 | 試合管理 | 試合日程、対戦相手、球場などの情報を管理する。 | G030-G034 |
| F040 | オーダー編成 | 試合ごとの打順、守備位置を設定する。 | G040 |
| F050 | スコア入力 | 試合のイニングスコア、打席結果（一球速報詳細含む）を入力する。 | G050, G060 |
| F060 | 成績集計 | 入力されたスコアから打率、防御率などを自動集計して表示する。 | G070 |
| F070 | メッセージ送受信 | チーム内または他チームとのメッセージ送受信を行う。 | G080-G082 |
| F080 | 公開設定 | チームや成績の公開/非公開を設定する。 | G013 |
| F090 | 問い合わせ | 運営への問い合わせを送信する。 | G100 |
